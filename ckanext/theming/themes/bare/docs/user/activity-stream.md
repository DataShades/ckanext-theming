# User Activity

View a user's activity stream.

## Overview

User activity page shows:
- Recent actions by user
- Dataset contributions
- Timeline of activities
- Interaction history

## URL Pattern

```
GET /user/activity/{id}
```

## Template

**File:** `templates/user/activity_stream.html`
