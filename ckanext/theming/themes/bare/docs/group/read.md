# Group Read

View group details and datasets.

## Overview

Group read page displays:
- Group information
- Group datasets
- Member information
- Group statistics

## URL Pattern

```
GET /group/{id}
GET /group/{name}
```

## Purpose

Allows users to:
- View group description
- Browse group datasets
- See group members
- Follow group
