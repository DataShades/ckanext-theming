# Group Section

This section covers all pages related to group management in CKAN.

## Overview

Groups in CKAN are used to:

- Categorize datasets thematically
- Create curated collections
- Enable community collaboration
- Organize content by topic

## Group vs Organization

| Feature     | Group                   | Organization        |
|-------------|-------------------------|---------------------|
| Purpose     | Thematic categorization | Dataset ownership   |
| Hierarchy   | Flat                    | Can be hierarchical |
| Privacy     | Usually public          | Can be private      |
| Permissions | Less restrictive        | Controls access     |
