# Statistics Page

View portal usage statistics and analytics.

## Overview

The statistics page displays:

- Total number of datasets over time
- Dataset revisions per week
- Most edited datasets
- Largest groups
- Top tags
- Users creating most datasets

![stats](../screenshots/stats.jpeg)

///

## URL Pattern

```
GET /stats
```

**Example:**
```
<<vars.site_url>>/stats
```

## Purpose

Allows users to:

- View portal growth trends
- Track dataset activity
- Identify popular content
- Understand usage patterns

## Statistics Sections

| Section                  | Description              |
|--------------------------|--------------------------|
| Total Datasets           | Dataset count over time  |
| Dataset Revisions        | Weekly revision count    |
| Most Edited Datasets     | Top edited datasets      |
| Largest Groups           | Groups by dataset count  |
| Top Tags                 | Most used tags           |
| Top Dataset Creators     | Users by dataset count   |

## Actions Available

| Action          | Description              |
|-----------------|--------------------------|
| View stats      | See statistics tables    |
| Navigate menu   | Jump to stat section     |
| View dataset    | Click to view dataset    |
| View group      | Click to view group      |
| View tag        | Click to view tag        |
| View user       | Click to view user       |
