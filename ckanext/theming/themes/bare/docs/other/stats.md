# Statistics Page

Site usage statistics and analytics.

## Overview

The stats page displays:
- Dataset statistics
- Organization statistics
- Group statistics
- User statistics
- Historical trends

## URL Pattern

```
GET /stats
```

**Example:**
```
<<vars.site_url>>/stats
```

## Purpose

The stats page allows users to:
- View portal statistics
- Track growth over time
- Compare categories
- Export statistics

## Template

**File:** `templates/ckanext/stats/index.html`

### Statistics Displayed

- Total datasets
- Total organizations
- Total groups
- Total users
- Total views/downloads
- Historical trends
