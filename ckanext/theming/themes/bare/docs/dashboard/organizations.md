# Organizations

View and manage your organizations.

## Overview

The organizations dashboard displays:

- Organizations you belong to
- Your role in each organization
- Quick access to organization pages

![organizations](../screenshots/dashboard-organizations.jpeg)

///

## URL Pattern

```
GET /dashboard/organizations
```

**Example:**
```
<<vars.site_url>>/dashboard/organizations
```

## Purpose

Allows users to:

- View organization memberships
- See roles and permissions
- Navigate to organizations
- Manage organizations (if admin)

## Actions Available

| Action           | Description              |
|------------------|--------------------------|
| View orgs        | See organization list    |
| Navigate to org  | Go to organization       |
| Manage org       | Edit organization        |
| View members     | See organization members |
