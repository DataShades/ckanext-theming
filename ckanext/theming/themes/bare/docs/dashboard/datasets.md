# Datasets

View and manage your datasets.

## Overview

The datasets dashboard displays:

- All datasets you've created
- Published datasets
- Draft datasets
- Quick edit actions

![datasets](../screenshots/dashboard-datasets.jpeg)

## URL Pattern

```
GET /dashboard/datasets
```

**Example:**
```
<<vars.site_url>>/dashboard/datasets
```

## Purpose

Allows users to:

- View all created datasets
- Access draft datasets
- Quick edit datasets
- Create new datasets

## Actions Available

| Action          | Description              |
|-----------------|--------------------------|
| View datasets   | See dataset list         |
| Edit dataset    | Modify dataset           |
| Create dataset  | Add new dataset          |
| Delete dataset  | Remove dataset           |
