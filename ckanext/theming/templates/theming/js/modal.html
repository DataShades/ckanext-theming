modal

Creates a modal dialog.

It accepts conent of the central part of modal as a first argument, which can be either text or DOM node. The object produced by the function has methods show and close that change visibility of the modal. Initially modal is hidden, and you need to call its show method to display the modal {%- call ui.util.call(ui.code, language="javascript", dedent=true) %} const modal = sandbox.ui.modal("Hello world") const btn = sandbox.ui.button("Show modal"); btn.addEventListener("click", () => modal.show()) document.getElementById(id).appendChild(btn) {%- endcall %}

The second argument is a title of the modal, and the third argument is an array of buttons placed in the footer of the modal. {%- call ui.util.call(ui.code, language="javascript", dedent=true) %} const alertBtn = sandbox.ui.button("Show alert"); alertBtn.addEventListener("click", () => alert(1)); const closeBtn = sandbox.ui.button("Close modal"); closeBtn.addEventListener("click", () => modal.close()); const footer = [alertBtn, closeBtn] const modal = sandbox.ui.modal("Hello world", "Modal title", footer) const btn = sandbox.ui.button("Show modal"); btn.addEventListener("click", () => modal.show()) document.getElementById(id).appendChild(btn) {%- endcall %}
