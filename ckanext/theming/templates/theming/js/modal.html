modal

Creates a modal dialog.

It accepts conent of the central part of modal as a first argument, which can be either text or DOM node. The second parameter is an object with options.

The object produced by the function has methods show and close that change visibility of the modal. Initially modal is hidden, and you need to call its show method to display the modal {%- call ui.util.call(ui.code, language="javascript", dedent=true) %} const modal = sandbox.ui.modal("Hello world", {dismissible: true}) const btn = sandbox.ui.button("Show modal"); btn.addEventListener("click", () => modal.show()) document.getElementById(id).appendChild(btn) {%- endcall %}

To add title and buttons in the footer to the modal, use its options title and actions. {%- call ui.util.call(ui.code, language="javascript", dedent=true) %} const alertBtn = sandbox.ui.button("Show alert"); alertBtn.addEventListener("click", () => alert(1)); const closeBtn = sandbox.ui.button("Close modal"); closeBtn.addEventListener("click", () => modal.close()); const actions = [alertBtn, closeBtn] const modal = sandbox.ui.modal("Hello world", {   title: "Modal title",   dismissible: true,   actions, }) const btn = sandbox.ui.button("Show modal"); btn.addEventListener("click", () => modal.show()) document.getElementById(id).appendChild(btn) {%- endcall %}

Other options available for modal:

  • dismissible: boolean, whether to add default close-button to the modal.
  • dismissLabel: text of the label for the button that closes the modal, e.g. "Close"
{%- call ui.util.call(ui.code, language="javascript", dedent=true) %} const modal = sandbox.ui.modal("Hello world", {   dismissible: true,   dismissLabel: "Close",   style: "danger", }) const btn = sandbox.ui.button("Show modal"); btn.addEventListener("click", () => modal.show()) document.getElementById(id).appendChild(btn) {%- endcall %}

To access existing modal, rendered with ui.modal macro, use getModal method available in the JS. This method accepts ID of the modal element. {%- call ui.util.call(ui.code, language="jinja2", dedent=true) %} {%- raw %} {% set modal_id = ui.util.id() -%} {{ ui.modal("Content", title="Title", dismissible=true, id=modal_id) }} <script>   const modal = sandbox.ui.getModal(`{{ modal_id }}`)   const btn = sandbox.ui.button("Show modal");   btn.addEventListener("click", () => modal.show())   document.getElementById(id).appendChild(btn) </script> {%- endraw %} {%- endcall %}
