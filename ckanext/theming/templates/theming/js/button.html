button

Creates a button with attributes required by theme.

It accepts conent of the button as a first argument, which can be either text or DOM node. The second argument is an object with options: {%- call ui.util.call(ui.list) -%} {%- call ui.util.call(ui.list_item) -%} type: can be button, submit, or reset. {%- endcall %} {%- call ui.util.call(ui.list_item) -%} style: any style supported by theme, e.g. primary, secondary, danger, etc. {%- endcall %} {%- endcall %}

{%- set id1 = ui.util.id() -%}

The function returns standard HTMLElement that can be added to the DOM. {%- call ui.util.call(ui.code, language="javascript", dedent=true) %} const btn = sandbox.ui.button("Click me", {"style": "danger"}); btn.addEventListener("click", () => alert(1)) document.getElementById(id).appendChild(btn) {%- endcall %}
