Standard

The popover is triggered by popover_handle, which often has no CSS applied to it. This allows developers to use any element as a popover handle, such as an icon, link, or button.

{%- set id = ui.util.id() -%} {%- call ui.util.call(ui.popover, title="Popover", id=id) -%} {{ lipsum(1) }}
{{ lipsum(1) }} {%- endcall %}

Usually {{ ui.popover_handle("[popover handle]", id=id) }} does not have custom appearance.

{%- call ui.util.call(ui.code, language="jinja2", dedent=true) -%} {%- raw %} {% set id = ui.util.id() -%} {%- call ui.util.call(ui.popover, title="Popover", id=id) -%} {{ lipsum(1) }} <hr/> {{ lipsum(1) }} {%- endcall %} <p> Usually {{ ui.popover_handle("popover handle", id=id) }} does not have custom appearance. </p> {%- endraw %} {%- endcall %}
