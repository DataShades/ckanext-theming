Search form

The search_form component is a recommended tool for rendering standard search form. It supports variety of parameters that partially overlap with parameters for CKAN's native search snippet

active_filters parameter can cause a problem if it receives fake value. To see how it looks, apply filter to the current page by opening {{ ui.link("this link", "?organization=gov") }}.
