Standard

datetime shows only the date by default, but you can also choose to show the time by setting the with_hours parameter to true.

{%- set now = ui.util.now() -%} Today: {{ ui.datetime(now) }}
Now: {{ ui.datetime(now, with_hours=true) }}
{%- call ui.util.call(ui.code, language="jinja2", dedent=true) -%} {%- raw %} Today: {{ ui.datetime(now) }} <br/> Now: {{ ui.datetime(now, with_hours=true) }} {%- endraw %} {%- endcall %}
