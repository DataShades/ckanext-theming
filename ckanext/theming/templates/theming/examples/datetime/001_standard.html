Standard

datetime shows only the date by default, but you can also choose to show the time by setting the with_hours parameter to true.

{%- call ui.util.call(show_example) -%} {%- raw %} {%- set now = ui.util.now() -%} Today: {{ ui.datetime(now) }}
Now: {{ ui.datetime(now, with_hours=true) }} {%- endraw %} {%- endcall %}
