Dismissible

Dismissible alerts can be dismissed by clicking on a close button. This behavior can be enabled by setting dismissible parameter to true.

{%- call ui.util.call(show_example) -%} {%- raw %} {{ ui.alert(lipsum(1, 1, 5), dismissible=true) }} {{ ui.alert(lipsum(1, 1, 5), dismissible=false) }} {%- endraw %} {%- endcall %}
