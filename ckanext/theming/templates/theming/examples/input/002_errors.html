Field errors

errors parameter of the input accepts list of error messages. When there are errors, the input will be styled with error styles and error messages will be shown below the input.

{{ ui.input(name="name", label="Name", errors=["Cannot be empty", "Letters only"]) }}
{%- call ui.util.call(ui.code, language="jinja2", dedent=true) -%} {%- raw %} {{ ui.input(name="name", label="Name", errors=["Cannot be empty", "Letters only"]) }} {%- endraw %} {%- endcall %}
