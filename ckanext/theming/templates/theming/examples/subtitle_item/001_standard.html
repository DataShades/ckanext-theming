Standard

This component modifies title of the page and usually reflects the state of breadcrumbs. It must be used inside subtitle block.

The subtitle_item inserts configured divider between items. When the initial parameter set to true, divider after such segment is not added.

{%- call ui.util.call(show_example) -%} {%- raw %} {{ ui.subtitle_item("Hello") }} {{ ui.subtitle_item("World") }} {{ ui.subtitle_item("CKAN", initial=true) }} {%- endraw %} {%- endcall %}
