Standard

The subtitle_item inserts configured divider between items. When the initial parameter set to true, divider after such segment is not added.

This component modifies title of the page and usually reflects the state of breadcrumbs: {%- call ui.util.call(ui.code, language="jinja2", dedent=true) -%} {%- raw %} {%- block subtitle -%} {{ ui.subtitle_item("Edit") }} {{ ui.subtitle_item("Dataset") }} {{ super() }} {%- endblock %} {%- block breadcrumb_content -%} {{ super() }} {{ ui.breadcrumb("Dataset", href="") }} {{ ui.breadcrumb("Edit", href="") }} {%- endblock %} {%- endraw %} {%- endcall %}
