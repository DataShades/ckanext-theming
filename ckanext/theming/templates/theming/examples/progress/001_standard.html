Standard

Progress bar is usually capped at 100, but you can set any value and max. If value is 0, the bar will be empty.

{{ ui.progress("Loading..", value=50) }} {{ ui.progress("Not loading", value=0) }} {{ ui.progress(value=10, max=10) }}
{%- call ui.util.call(ui.code, language="jinja2", dedent=true) -%} {%- raw %} {{ ui.progress("Loading..", value=50) }} {{ ui.progress(value=0) }} {{ ui.progress(value=10, max=10) }} {%- endraw %} {%- endcall %}
