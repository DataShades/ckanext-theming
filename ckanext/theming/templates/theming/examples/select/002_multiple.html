Multiple

The select can accept a list of selected options when it's rendered with multiple flag. But this is an optional feature, because it was not available in native CKAN's implementation.

{%- set options = [{"value":"a"}, {"value":"b"}, {"value":"c"}, {"value":"d"}] -%} {{ ui.select(label="List", multiple=true, options=options, selected=["b", "c"]) }}
{%- call ui.util.call(ui.code, language="jinja2", dedent=true) -%} {%- raw %} {%- set options = [{"value":"a"}, {"value":"b"}, {"value":"c"}, {"value":"d"}] -%} {{ ui.select(label="List", multiple=true, options=options, selected=["b", "c"]) }} {%- endraw %} {%- endcall %}
