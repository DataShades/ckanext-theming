Using HTML tags

As long as you use the appropriate HTML structure and classes, you can create tables using native HTML tags.

{%- call ui.util.call(ui.table) -%} Key Value {%- for value in ["A", "B", "C"] -%} {{ loop.index }} {{ value }} {%- endfor %} {%- endcall %}
{%- call ui.util.call(ui.code, language="jinja2", dedent=true) -%} {%- raw %} {%- call ui.util.call(ui.table) -%} <thead>   <tr>     <th>Key</th>     <th>Value</th>   </tr> </thead> <tbody> {%- for value in ["A", "B", "C"] -%} <tr>   <td>{{ loop.index }}</td>   <td>{{ value }}</td> </tr> {%- endfor %} </tbody> {%- endcall %} {%- endraw %} {%- endcall %}
