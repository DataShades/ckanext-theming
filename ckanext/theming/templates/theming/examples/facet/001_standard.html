Facet section

A facet section, typically used in a sidebar for filtering search results. It contains facet_list with series of facet components.

{%- call ui.util.call(ui.filters) -%} {{ ui.facet_section(lipsum(1), title="Nothing") }} {%- set items = [ {"name": "one", "count": 1}, {"name": "ten", "count": 10, "active": true}, {"name": "hundreed", "count": 100}, ] -%} {%- call ui.util.call(ui.facet_section, title="Something", items=items, has_more=true) -%} {%- call ui.util.call(ui.facet_list) -%} {%- for item in items -%} {{ ui.facet( "Facet " ~ item.name, key="", value=item.name, count=item.count, active=item.active) }} {%- endfor %} {%- endcall %} {%- endcall %} {%- endcall %}
