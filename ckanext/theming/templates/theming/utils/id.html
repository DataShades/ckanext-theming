ui.util.id

Generate a unique identifier.

If value is provided, a UUID5 based on the value is generated, otherwise a random UUID4 is generated.

Call without arguments produces a different UUID every time: {%- call ui.util.call(ui.code, dedent=true, language="jinja2") -%} {%- raw %} {{ ui.util.id() }} {{ ui.util.id() }} {%- endraw %} {%- endcall %}

{{ ui.util.id() }}
{{ ui.util.id() }}

When called with value, which is first positional argument, the same UUID is returned for every call: {%- call ui.util.call(ui.code, dedent=true, language="jinja2") -%} {%- raw %} {{ ui.util.id("test") }} {{ ui.util.id("test") }} {%- endraw %} {%- endcall %}

{{ ui.util.id("test") }}
{{ ui.util.id("test") }}
