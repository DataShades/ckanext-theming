ui.util.attrs

Helper method to render HTML attributes from a dictionary with additional macro arguments.

This method takes a dictionary that contains optional attrs element, and an optional dictionary of default attributes. It combines the two dictionaries, giving precedence to the provided attributes, and renders them as a string of HTML attributes.

Example: {%- call ui.util.call(ui.code, dedent=true, language="jinja2") -%} {%- raw %} {% macro button(content) %} {% set defaults = {"class": "btn btn-primary"} %} <button {{ ui.util.attrs(kwargs, defaults) }}>{{ content }}</button> {% endmacro %} {{ ui.button("Click me!", attrs={"name": "action"}) }} {%- endraw %} {%- endcall %}

{%- filter forceescape -%} ! {%- endfilter %}

Note, this function expects dictionary with attrs element, not the attrs itself. For example, following example is incorrect: {%- call ui.util.call(ui.code, dedent=true, language="jinja2") -%} {%- raw %} {% macro button(content) %} <button {{ ui.util.attrs(kwargs.attrs) }}>{{ content }}</button> {% endmacro %} {%- endraw %} {%- endcall %}

Instead, the whole kwargs must be passed into function. Or, if kwargs is not available, you can wrap attrs into a dictionary. Here are correct versions: {%- call ui.util.call(ui.code, dedent=true, language="jinja2") -%} {%- raw %} {% macro button(content) %} <button {{ ui.util.attrs(kwargs) }}>{{ content }}</button> {% endmacro %} {% macro button(content) %} <button {{ ui.util.attrs({"attrs": kwargs.attrs}) }}>{{ content }}</button> {% endmacro %} {%- endraw %} {%- endcall %}

The reason for this is that ui.util.attrs expects to receive the whole set of arguments, so it can extract the attrs element and combine it with defaults. If only attrs is passed, the function will not be able to access other arguments that might be needed for default values or other logic.

One of the features relying on other values of kwargs is attribute shortcuts. Every attribute with data-, aria-, hx-, and on prefix can be specified inside separate item of kwargs. Example {%- call ui.util.call(ui.code, dedent=true, language="jinja2") -%} {%- raw %} {{ ui.button("Click me!", data={"id": "123"}, aria={"label": "Button"}, hx={"boost": "true"}, on={"click": "alert(1)"}) }} {%- endraw %} {%- endcall %}

{%- filter forceescape -%} {%- endfilter %}

All these shortcuts can be used together with attrs, and specific attributes will have higher precedence than attrs. {%- call ui.util.call(ui.code, dedent=true, language="jinja2") -%} {%- raw %} {{ ui.button("Click me!", attrs={"data-id": "123", "aria-label": "Button"}, data={"id": "456"}, aria={"label": "Overridden"}) }} {%- endraw %} {%- endcall %}
