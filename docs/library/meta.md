# Meta Macros

## `header_block`

Renders a header block. This is a block macro.

### Arguments

*   `title` (string): The main title to display. Defaults to `None`.
*   `subtitle` (string): The subtitle to display. Defaults to `None`.
*   `logo` (string): Path to the logo image. If provided, overrides title/subtitle. Defaults to `None`.

## `footer_block`

Renders a footer block. This is a block macro.

### Arguments

None

## `account_block`

Renders an account-related block. This is a block macro.

### Arguments

None
